ADDITIONAL INFORMATION

- GS FILE:
This file calls the data from the spreadsheet to appear on the website.
- .HTML FILE:
This is the main file that you will edit if you want to make a change to the code. Here are instructions on how to make a change and have it reflect on the website (deploy your website).

Instructions on Editing App Script:
1. Finish your edits. Press the printer button to “Save Project to Drive.” Go to the Code.GS file and press “Run” to catch any errors.
2. Press the right side of the blue “Deploy” button in the top right corner to get to the drop-down. Select “Manage Deployments.”
3. Press the pencil button in the top right. Go to the Version Drop-down and select “New Version.” Add a description if you’d like. 
4. Press the “Deploy” button and you’re done!
